# nike_shop

A store shopping clone for nike sneakers that has features such as shopping cart, favorites, online shopping, on-site shopping and user comments.

# Packages used

1- bloc

2- cached_network_image

3- cupertino_icons

4- dartz

5- dio

6- equatable

7- expandable

8- flutter_bloc

9- flutter_localizations

10- flutter_spinkit

11- get_it

12- hive

13- hive_flutter

14- persian_number_utility

15- shared_preferences

16- smooth_page_indicator

17- uni_links

18- url_launcher

19- zarinpal
